# QA Automation Assignment - Harin Patel

## Overview
This project contains end-to-end automation test cases using Selenium, Java, and TestNG.

## Tech Stack
- Java
- Selenium WebDriver
- TestNG
- Maven
- Chrome (Headless)

## Test Scenarios
- File Upload
- Drag and Drop
- Carousel Validation
- Calendar Validation
- Amazon Search

## How to Run

1. Clone repository
git clone https://github.com/<your-username>/Harin-Patel.git

2. Navigate to project
cd Harin-Patel

3. Run tests
mvn test

## Notes
- Uses Page Object Model (POM)
- Uses explicit waits for stability
- Runs in headless mode
